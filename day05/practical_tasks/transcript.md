Hello, my name is Sehar Andleeb and I am an AI Engineer Intern
at Xeven Solutions. Today I will explain the difference between
regression and classification in Machine Learning.

Both regression and classification are types of Supervised
Learning — meaning the model learns from labeled data. The key
difference is in what they predict.

Regression predicts a continuous number. Think of it like
estimating the price of a house. You give the model information
like the size of the house, number of bedrooms, and location,
and it gives you back a number — for example 250,000 dollars.
The output can be any number in a range.

Classification on the other hand predicts a category. Think of
it like a spam filter in your email. You give the model an email
and it tells you — this is either Spam or Not Spam. The output
is always one of a fixed set of categories.

A simple way to remember the difference:
- If the answer is a number — it is Regression
- If the answer is a category — it is Classification

Real world examples of Regression:
- Predicting house prices
- Forecasting tomorrow's temperature
- Estimating a student's exam score

Real world examples of Classification:
- Detecting spam emails
- Diagnosing whether a patient has a disease
- Recognizing whether an image contains a cat or a dog

That is the difference between Regression and Classification.
Thank you for listening.
